프로그레스바 Edit on GitHub

프로그레스바를 설정하는 가장 쉬운 방법은 앱에서 상속받은 속성 설정할 때 ProgressBarDrawable 클래스를 사용하면 됩니다. java .setProgressBarImage(new ProgressBarDrawable())

이것은 Drawee의 하단을 따라 어두운 파란색 네모 프로그레스바를 보여줍니다.

당신만의 프로그레스바 정의하기

class CustomProgressBar extends Drawable {
   @Override
   protected void onLevelChange(int level) {
     // 0-10,000까지의 스케일 레벨
     // 10,000은 완전히 다운 받음 의미

     // 프로그레스에 따른 drawable의 모양을
     // 바꾸는 로직
   }
}
